# 계층별 인식 조사 — 데이터 분석 그래프

원본: `민원과 교육활동 위축에 관한 계층별 인식 조사.csv` (응답 N=113 — 교사 31·학부모 69·학생 13)

> 이 설문은 교사/학부모/학생 폼이 한 시트에 합쳐져 있어 같은 "Q번호"라도 계층마다 질문이 다르다.
> 계층 비교는 **의미가 같은 문항끼리 매핑**해 수행했다.
> (핵심: 교사 Q12 · 학부모 Q7 · 학생 Q5 = 모두 *"과도한 민원이 교사를 위축·부담시킨다"*)

## 핵심 결론

- **"과도한 민원이 교사를 위축시킨다"** 동의: 교사 4.65(97%) · **학부모 4.30(90%)** · 학생 4.00(69%) → 계층을 가로지르는 합의.
- **학부모의 두 얼굴**: "대부분 민원은 정당"엔 52%만 동의(3.28)하지만, "일부 과도민원이 문제"엔 81~90% 동의(4.14~4.30). → 민원 전체 옹호가 아니라 *'일부 악성'을 분리해 문제로 인식*.
- 공통문항 중 **'과도민원 부담'에서 교사·학부모 격차가 가장 작다**(4.65 vs 4.14).
- 교사 61%가 지난 1년 도 넘는 민원 1회 이상 경험. 유형은 생활지도 항의 > 퇴근후 개인연락 > 성적 이의.
- 학부모 58%가 주변 과도민원을 직접 목격(10%)하거나 전해 들음(48%).
